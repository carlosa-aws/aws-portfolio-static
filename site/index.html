About This Project

This portfolio was built to showcase hands-on AWS and Terraform skills. The frontend is served securely through CloudFront, while a Lambda function updates and returns the visitor count stored in DynamoDB.

Infrastructure is managed as code with Terraform, making the deployment repeatable and easier to maintain.

Architecture Highlights

  • Static website hosted in Amazon S3
  • Content delivered through Amazon CloudFront
  • Visitor counter powered by AWS Lambda
  • Count stored in Amazon DynamoDB
  • Infrastructure provisioned with Terraform
