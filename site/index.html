About This Project

This portfolio website was built to demonstrate hands-on AWS cloud engineering skills. The frontend is served through Amazon CloudFront from a private S3 bucket, while a serverless backend updates the visitor counter using AWS Lambda and DynamoDB.

All infrastructure is provisioned using Terraform to demonstrate Infrastructure as Code and reproducible cloud deployments.

Architecture Highlights

  • Static website hosted in Amazon S3
  • Content delivered globally via CloudFront CDN
  • Visitor counter powered by AWS Lambda
  • Visitor count stored in DynamoDB
  • Infrastructure provisioned using Terraform

Technology Stack

Terraform AWS S3 CloudFront Lambda DynamoDB Python HTML CSS JavaScript

What I Practiced

  • Infrastructure as Code using Terraform
  • Deploying secure static websites on AWS
  • Serverless backend development with Lambda
  • Integrating DynamoDB with API Gateway
  • Cloud architecture documentation and GitHub portfolio presentation
